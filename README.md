# TechNova Solutions — CompanyPortfolio

Premium Enterprise IT company portfolio built with HTML5, CSS3, Vanilla JavaScript and Bootstrap 5.3.

Structure:

- `index.html` — Landing site with full sections
- `css/style.css` — Core styles
- `css/responsive.css` — Responsive overrides
- `js/script.js` — Core interactions
- `js/darkmode.js` — Theme toggle
- `js/counter.js` — Animated counters
- `js/typing.js` — Hero typing animation
- `images/` — SVG / image assets

How to use:

Open `index.html` in a browser. No build required.

Notes:
- The contact form in this static project simulates submission (no backend).
- AOS is used for viewport animations (included via CDN).
